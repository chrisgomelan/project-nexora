# Nexora WordPress Child Theme Installation Guide

![Nexora Banner](screenshot.png)

This guide provides steps to install and activate the Nexora child theme, a premium, high-performance theme built on the **Twenty Twenty-Five** base theme.

## 🛠️ Prerequisites

Before installing, ensure that the parent theme **Twenty Twenty-Five** is installed in your WordPress environment.

## 📦 Installation Steps

1. **Upload the Theme**: 
   Upload the `child-nexora` folder to your WordPress installation's `/wp-content/themes/` directory via FTP or your hosting file manager.
   
2. **Activate the Theme**:
   - Log in to your WordPress Dashboard.
   - Navigate to **Appearance > Themes**.
   - Locate the **Nexora** theme.
   - Click the **Activate** button.

3. **Verify Installation**:
   Once activated, the theme will automatically enqueue the necessary assets and register the custom Gutenberg blocks included in the package.

## 📂 Project Structure

- `assets/` - Contains CSS, JavaScript, and image assets.
- `blocks/` - Custom Gutenberg block definitions and logic.
- `parts/` - HTML template parts for the Site Editor.
- `templates/` - Custom page templates.
- `functions.php` - Core theme logic and asset enqueuing.
- `style.css` - Theme metadata and global CSS overrides.
- `theme.json` - Global styles and settings configuration.
