# Nexora WordPress Child Theme

![Nexora Banner](screenshot.png)

A premium, high-performance WordPress child theme built on the **Twenty Twenty-Five** base theme. Nexora is designed for professional brands that demand a bold, corporate identity with modern, cinematic interactions.

## 🚀 Overview

Nexora combines the flexibility of block-based editing with custom-engineered Gutenberg blocks and a sophisticated SEO-first architecture. It features a curated design system centered around a "Nexora Navy" identity, utilizing high-end typography and smooth, scroll-triggered animations.

## 🛠️ Technology Stack

- **Core**: WordPress (Block Theme Architecture)
- **Parent Theme**: Twenty Twenty-Five
- **Design System**: Vanilla CSS with custom property tokens (`theme.json`)
- **Typography**: Poppins (Headings) & Inter (Body/UI)
- **Icons**: Font Awesome 6.5.1
- **Animations**: Purpose-built JavaScript for scroll-driven interactions

## ✨ Key Features

### 🧩 Custom Gutenberg Blocks
Nexora includes a suite of custom-registered blocks for seamless page building:
- **Hero Section**: A high-impact entry section with cinematic text reveals.
- **Scroll Curtain Reveal**: A premium wrapper block that applies a sophisticated "curtain" animation to its children.
- **Content Split**: Flexible side-by-side layouts for features and messaging.
- **Testimonials**: A sleek, stylized block for showcasing social proof.
- **CTA Section**: High-conversion call-to-action blocks.
- **Nexora Footer**: A fully integrated, semantic footer with dynamic navigation support.

### 🎨 Design System & UI
- **Custom Button Variations**: 
  - `Nexora Primary`: Solid Navy with rounded corners and optimized padding.
  - `Nexora Secondary`: Outline style for secondary actions.
- **Rich Aesthetics**: Glassmorphism effects, smooth gradients, and micro-animations for enhanced engagement.
- **Responsive-First**: Meticulously crafted for mobile, tablet, and desktop experiences.

### 🔍 SEO & Performance
- **Automated Meta Tags**: Dynamic generation of meta descriptions, keywords, and author tags.
- **Structured Data**: Built-in JSON-LD for `Organization` and `LocalBusiness` to boost SERP presence.
- **Semantic HTML**: Proper heading hierarchy and ARIA-compliant navigation.
- **Optimized Assets**: Lazy-loading support and minified font delivery via Google Fonts.

## 📦 Installation

1. Upload the `child-nexora` folder to your WordPress installation's `/wp-content/themes/` directory.
2. Navigate to **Appearance > Themes** in the WordPress Dashboard.
3. Locate **Nexora** and click **Activate**.

*Note: Ensure the parent theme **Twenty Twenty-Five** is installed.*

## 📂 Directory Structure

```text
child-nexora/
├── assets/          # CSS, JS, and Images
├── blocks/          # Custom Gutenberg block definitions
├── parts/           # HTML template parts for the Site Editor
├── templates/       # Custom page templates
├── functions.php    # Theme logic and asset enqueuing
├── style.css        # Theme metadata and global overrides
├── theme.json       # Global styles and settings configuration
```
